# Day 4 — Pillar 2: Reviews & Referrals
**Goal:** build the second service — an automation that wins reviews (with a raffle + 1–5
routing) and turns happy customers into referrals.

**Do:**
1. Run `prompts/4-reviews-referrals.md` (with the raffle/incentive).
2. Save SMS+email messages to `pillars/reviews-referrals.md`.
3. Wire the routing: 4–5 → Google review link; 1–3 → private feedback form; after a review →
   thank + ask referral (shareable link, extra raffle entries).
4. Compliance: incentivise honest feedback, not positive reviews; consent + opt-out.

**Output asset:** `pillars/reviews-referrals.md`.
