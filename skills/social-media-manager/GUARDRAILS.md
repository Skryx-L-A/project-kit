# GUARDRAILS — enforced

The `brand-safety-reviewer` checks every action; violations block the publish/spend.

## Platform ToS / authenticity
- Respect each platform's **automation + rate limits**. No behavior that mimics human bulk
  actions where banned (mass follow/DM/connect).
- **No fake engagement** — no bought likes/followers, no bot networks, no engagement pods.
- **No vote manipulation, sock-puppets, or astroturfing** (esp. Reddit). One account = one
  honest voice.
- **No mass-DM spam.** Outreach is gated + personalized.

## Reddit-specific
- Read + obey each subreddit's rules; hold the ~90/10 self-promo ratio; correct flair; no
  ban evasion; participate as a real member.

## Honesty / disclosure
- **FTC `#ad` / paid-partnership disclosure** wherever there's a material connection.
- No false claims, fake scarcity, fake testimonials, or misleading before/after.
- Creatives: own/licensed + rights-cleared; **no deepfakes of real people without consent**.

## Approval + secrets
- **Nothing public or money-spending without the configured approval gate** (`AUTONOMY.md`).
- Never commit account secrets/tokens — `.env` only (gitignored), `.env.example` documents names.
- No real personal data of third parties committed to a public repo.

## Footer
- Default **no emojis** unless the brand voice explicitly calls for them (ask first).
- Commits under the user's own name only (Skryx-L-A); never add Claude as co-author.
