# Platform — X / Twitter
- **Access**: X API v2 (OAuth2; paid tiers for write) or browser automation fallback.
- **Format**: single posts (≤280 default), threads, media (images/video/GIF), polls. Hook in
  the first line; one idea per post; threads for depth.
- **Posting/scheduling**: API create-tweet; schedule via n8n or native. Space posts; reply to
  build reach.
- **Engagement**: reply, quote, like; join conversations early. Drafts → gate → publish.
- **ToS**: no automated bulk/duplicate posting, no fake engagement, rate limits. Disclose #ad.
