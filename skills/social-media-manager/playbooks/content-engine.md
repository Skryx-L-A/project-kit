# Playbook — Content engine

A repeatable loop, not random posting.

1. **Plan** — from goals + pillars, set weekly themes; fill a calendar (`calendar/`).
2. **Create** — draft copy (copywriter) + generate creatives (ad-creative-generator via
   higgsfield). One core idea → repurpose per platform (a thread → a reel → a carousel → a
   Reddit text post written natively, not cross-posted verbatim).
3. **Schedule** — queue into `queue/` (review) then native scheduler / n8n; space + best-time.
4. **Publish** — through the approval gate per `AUTONOMY.md`.
5. **Engage** — reply to comments/DMs in the first hour (reach window).
6. **Measure** — weekly: what performed, why; double down on winners → `metrics.md`.

Repurposing rule: adapt to each platform's native form + rules (esp. Reddit — never paste the
same promo across subs).
