# Mode — Summarize

Compress threads and the inbox into something actionable. Read-only.

## Thread summary
- TL;DR (2–3 lines): what it's about, current state, the ask.
- **Decisions made** + **open questions**.
- **Action items** with owner + due date if stated.
- "**Needs a reply from you**" — yes/no + why.

## Inbox catch-up (after time away)
- Group by label/priority; one line per important thread.
- A ranked "reply queue": what to answer first and the suggested gist.
- Flag anything time-sensitive that slipped (deadlines passed/imminent).

## Style
Factual, no filler. Quote the source line for any claim that matters. Mark *inferred* vs
*stated*. Never fabricate a detail to make a summary cleaner.
